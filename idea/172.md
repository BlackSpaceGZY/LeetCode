## 172. 阶乘后的零



### 题目描述

给定一个整数 n，返回 n! 结果尾数中零的数量。

```
示例 1:

输入: 3
输出: 0
解释: 3! = 6, 尾数中没有零。
示例 2:

输入: 5
输出: 1
解释: 5! = 120, 尾数中有 1 个零.
```


说明: 你算法的时间复杂度应为 O(log n) 。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/factorial-trailing-zeroes



### 类型

数学



### 题解

花了很长时间，想过

哈希表存储每一次的值，超时；

用加法遍历，超时；

看到时间复杂度就应该是**乘法**

类似于之前做阶乘、除法的题一样的思路；不过这个题加了**规律**；



### 代码

```python
class Solution:
    def trailingZeroes(self, n: int) -> int:
    	sum1 = 0
    	while n >= 5:
    		num, i = 0, 5
    		while i <= n:
    			num = 1 + num * 5
    			i *= 5
    		n -= (i // 5)
    		sum1 += num
    	return sum1
```
